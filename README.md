# <img src="logo.png" width="60" align="center" /> My FIRE plan (个人资产管理系统) 

一个专为 FIRE (Financial Independence, Retire Early) 追求者打造的**极简个人资产管理系统**。

它不记流水，只记快照。它不追求复杂的会计平衡，只关注净值的长期增长。数据完全私有，部署在本地。


## 👋 为什么做这个软件？

以前我也像很多人一样，每天盯着股票软件，看到涨了就想追，看到跌了就想割，每天都在记录琐碎的流水，结果不仅没赚到钱，还搞得自己很焦虑。

后来我意识到，要想在市场上活得久并实现 FIRE (财务自由)，我必须对抗自己的人性。于是我写了这个工具，它强制我执行一套**“笨”**但有效的策略：

1. **极简记录 (余额快照法)**：
我不再折磨自己去记每一笔买菜钱或者股票交易流水。我**每周或每月**只花10分钟，把各个账户的总金额填进去。软件只关心**“现在有多少”**，不关心“怎么来的”。
2. **关注占比，而非涨跌**:
这个软件**不能**记录不同资产的累计收益，**没有**实时行情刷新功能（故意的，反正也不联网）。我不想看到某只股票今天涨了 5%，我只想知道我的**“风险敞口”**是不是太大了？比如“高风险资产”是不是超过了 60%？
3. **机器指挥操作 (再平衡)**:
我会设定一个理想的资产比例。以此为依据指导我调仓。

**我利用这个工具，关注10年、20年的复利增长，而不是今天的涨跌**

**当然了，一定要选好投资标的，别买了潘多拉星球概念ETF之后，杰克萨利叛变了，导致这个ETF一跌再跌，你还不断加仓，最后全给纳威人随份子了。**

## 🚀 极速启动 (PC端)

不用搞复杂的服务器，在自己电脑上就能跑。

### 1. 安装

```bash
git clone https://github.com/chaishilin/myFirePlan.git
cd myFirePlan
pip install -r requirements.txt
```

### 2. 初始化 (含演示数据)

为了方便大家体验，我写了个脚本生成了一套“模拟数据”（包含几个有趣的科幻资产），让你不用填数据也能看到效果。

```bash
python init_db.py    # 建表
python init_demo.py  # 注入演示数据
```

### 3. 启动

```bash
streamlit run app.py
```

启动后浏览器会自动打开。默认账号：`demo` (无需密码)。

## 📸 功能图解 (我是怎么用的)

### 图 1：风险与水位监控 + 净值走势
![图1](/picture/pic1-c.png "图1")


* **我看什么**：我不看今天赚了多少，我只看**“当前回撤”**。只要净值曲线大趋势是向上的，中间的波动我就当没看见。
* **水位线**：这里会显示我距离“历史最高资产”跌去了多少。控制回撤，比追求暴利更重要。

### 图 2：结构化趋势详细对比
![图2](/picture/pic2-c.png "图2")

* **我看什么**：通过折现图，看我的资产结构有没有**“漂移”**。
* **心得**：如果哪天发现“高风险资产”占比突然变大了，即使在赚钱，我也会警惕——因为这意味着我的风险敞口失控了。

### 图 3：资产与标签管理 (地基)
![图3](/picture/pic3-c.png "图3")

* **怎么用**：把你的美股、A股、基金、甚至外星币都录进去。
* **打标签**：这是灵魂！一定要给资产打上`风险偏好`（稳健/激进）或`投资位面`（美股/A股）的标签。后面的所有分析都依赖这个。

### 图 4：每日资产快照录入 (极简记账)
![图4](/picture/pic4-c.png "图4")

* **怎么用**：不用每天填！想起来的时候（比如发工资日、月底），把各个APP里的总资产填一下就行。

### 图 5：定投计划与未来现金流
![图5](/picture/pic5-c.png "图5")

* **我看什么**：这就像个**“天气预报”**。它告诉我未来30天，我那些自动定投计划一共要扣掉多少钱。
* **作用**：防止我把手头的现金花光了，导致定投断供。
